# Detection of Calving Events at Perito Moreno Glacier 
# Using Deep Learning on Time-Lapse Imagery

## Overview
This repository contains the code associated with the paper:

> [Author et al. (year). Detection of Calving Events at Perito Moreno Glacier Using Deep Learning on Time-Lapse Imagery. Journal of Geophysical Research: Earth Surface. DOI]

We develop a lightweight 2D-CNN for binary classification of calving 
events from terrestrial time-lapse imagery at the Brazo Rico front of 
Perito Moreno Glacier, Patagonia. The model is compared against VGG16, 
ResNet-50, and MobileNet using standard classification metrics and 
computational cost indicators.
